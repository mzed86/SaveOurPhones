# SaveOurPhones

Give your old iPhone a second life. A collection of lightweight Progressive Web Apps (PWAs) designed to run on older iOS devices.

## Why?

Millions of perfectly functional iPhones get discarded each year. These devices still have great screens, batteries, and sensors - they just can't run the latest apps. SaveOurPhones provides simple, useful apps that work offline and breathe new life into old hardware.

## Apps

### Time & Productivity
- **Bedside Clock** - Large display clock with alarm
- **Pomodoro** - Focus timer with breaks
- **Kitchen Timer** - Multiple countdown timers
- **Metronome** - Keep the beat
- **Habit Tracker** - Build daily habits

### Wellness & Utility
- **Breathing** - Guided breathing exercises
- **Notes** - Quick offline notes
- **Torch** - Flashlight & screen light
- **Sound Meter** - Measure noise levels
- **Level** - Bubble level tool

### Tools & Reference
- **Unit Converter** - Convert measurements
- **Tally Counter** - Count anything
- **Color Picker** - Pick and save colors
- **Bus Times** - London TfL arrivals

## Features

- **Lightweight** - Each app is under 500KB
- **Offline-first** - Works without internet after first load
- **No App Store** - Install directly from Safari
- **Battery efficient** - Minimal resource usage
- **Touch optimized** - Large tap targets, intuitive gestures

## How to Install

1. Open the site in Safari on your iPhone
2. Browse to any app
3. Tap the Share button
4. Select "Add to Home Screen"
5. The app now works offline like a native app

## Technical Details

- Vanilla HTML, CSS, and JavaScript (no frameworks)
- Service Workers for offline caching
- Web App Manifests for home screen installation
- Supports iPhone 5s and later (iOS 12+)
- Dark theme optimized for OLED screens
- Safe area insets for notched devices
